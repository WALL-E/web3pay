# Web3Pay

Web3 User Authentication Service

## Supported wallets

- solana/phantom

## Login step

1. /getUserId
   ```
   curl -X 'POST' \
    'http://127.0.0.1:3000/getUserId' \
    -H 'Content-Type: application/json' \
    -d '{
      "address": "5oNDL3swdJJF1g9DzJiZ4ynHXgszjAEpUkxVYejchzrY"
    }'

   {
     "result":"c9fe7bf01a33e35c"
   }
   ```
2. /getUserToken
